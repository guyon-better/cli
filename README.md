# @guyon/cli
## 安装

### npm

``` 
$ npm install -g @guyon/cli
```

### yarn
```
$ yarn global add @guyon/cli
```

## 使用

### 初始化项目
```
$ guyon init <project-name>
or
$ guyon i <project-name>
```

### 生成模块
```
$ guyon generate <module-name> <destination-path>
or
$ guyon g <module-name> <destination-path>
```
